# Interfaz de el conductor

#### Requisitos para ver el proyecto:
1. Tener node instalado
2. Tener un editor de codigo instalado

#### Pasos para ver el proyecto
1. Copien el repositorio
2. Abran el proyecto en su editor de codigo de preferencia
3. Abran una nueva termianl en la direccion de el proyecto
4. Ejecuten el siguiente comando:
  `npm run dev`

> Nota importante: 
En caso de que esten en windows y les marque error a la hora de ejecutar el comando. Ejecutar siguiente comando:
`Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`, poserior a eso ejecuten `npm run dev`
